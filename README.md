# Visual Secret Sharing Scheme (VSSS)


**Recommended Environment:** [Google Colab](https://colab.research.google.com/github/Izucchi-channel/vsss/blob/vsss.ipynb)

If you are using Google Colab, no additional dependencies need to be installed.

If you are running it locally, use `python app.py`.

**Created by:** [@izucchi-tech](https://www.youtube.com/@izucci-tech)

## How to Use

Upload the video file you want to convert using the input field.

Since the generation process can take some time, you can generate a preview for a specified number of seconds.

A detailed explanation of how the scheme works will be published in a future video.

Reference: https://dl.acm.org/doi/10.1145/2935618

# 視覚復号型秘密分散暗号

**推奨実行環境：** [Google Colab](https://colab.research.google.com/github/Izucchi-channel/vsss/blob/vsss.ipynb)

Google Colabで実行する場合、追加の依存関係をインストールする必要はありません。

ローカルで実行する場合は、`python app.py`を使用します。

**制作:** [@izucchi-tech](https://www.youtube.com/@izucci-tech)

## 使用方法

変換したい動画ファイルを、入力欄にアップロードしてください。

生成には時間がかかるため、指定した秒数間のプレビューの生成ができます。

詳しい仕組みについては、後日解説動画を出す予定です。

参考文献：https://dl.acm.org/doi/10.1145/2935618
